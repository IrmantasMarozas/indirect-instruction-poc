What is Indirect Injection?

When AI agents retrieve external content, embedded instructions can manipulate their behavior without the user's knowledge.

Learn More

Check the README.md for detailed attack vectors, defense strategies, and research findings.

Defense Strategies

Input sanitization, content validation, sandboxed processing, and user confirmation for sensitive actions.

Test Scenarios

  • Technical documentation poisoning (Kubernetes, DevOps guides)
  • Hidden canary tokens in search results
  • SEO-optimized injection pages
  • Reference material manipulation
