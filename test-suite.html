Automated testing for indirect instruction injection patterns

What This Does

This test suite demonstrates how well the detection engine catches different types of injection attacks. Click "Run All Tests" below to watch 12 test cases execute automatically.

You'll see:

  • Green (Pass): Detection worked correctly - caught malicious content or correctly identified clean content
  • Red (Fail): Detection missed an attack or falsely flagged clean content
  • Confidence scores showing how certain the detector is
  • Pattern counts showing what suspicious elements were found

For developers: This shows baseline detection accuracy. You can integrate your own detection engine to compare performance.
